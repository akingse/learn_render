## Games101作业

配置eigen和opencv

```
指定路径 
C:\Users\wangk\source\repos\TPL>
下载vcpkg
git clone https://github.com/Microsoft/vcpkg.git
执行
cd .\vcpkg\
.\bootstrap-vcpkg.bat 
安装 opencv
.\vcpkg.exe install opencv:x64-windows

集成到vs环境
.\vcpkg integrate install
.\vcpkg integrate remove //删除
```
